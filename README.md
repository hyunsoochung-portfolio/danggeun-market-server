# 🍊감귤 마켓

## About The Project

본 레포지토리는 **와플스튜디오 23.5기 9조**의 중고거래/경매 프로젝트 **감귤 마켓**입니다.
**빈티지 물품을 경매로 사고 팔 수 있는 중고거래 사이트**를 목표로 하며, 기본적인 중고 거래 기능에 더해 실시간 채팅과 경매 시스템을 제공합니다.

---

### Frontend

* 설한동, 정현수

### Backend

* 부민성, 장우혁, 송민혁

---

## Tech Stack

* **Backend**: FastAPI
* **Database**: MySQL
* **Storage**: AWS S3
* **Infra**: Docker, Nginx
* **CI/CD**: GitHub Actions

---

## Workflow

* 기본 작업 브랜치: `dev`
* 기능 개발: `feature/<function>` 브랜치
* 기능 완료 후 `dev` 병합 및 디버깅
* 안정화 후 `main` 브랜치 병합

---

## Features

### 1. 인증 / 회원 관리

* 이메일 기반 회원가입 / 로그인
* 비밀번호 해싱 및 중복 검사
* JWT 인증 (Access / Refresh)
* OAuth2 소셜 로그인 (확장)

### 2. 게시물 (중고 거래)

* 게시물 CRUD
* 카테고리 / 지역 기반 필터링
* 이미지 다중 업로드
* 페이지네이션 지원

### 3. 지역 설정

* 사용자 기본 지역 설정
* 지역 기반 게시물 노출

### 4. 좋아요 / 관심 목록

* 게시물 좋아요 등록 / 취소
* 관심 게시물 목록 조회

### 5. 가상 화폐 거래

* 포인트 지갑 관리
* 거래 상태 관리
* 트랜잭션 기반 정합성 처리

### 6. 채팅

* 게시물 기반 1:1 채팅
* WebSocket 실시간 메시지
* 채팅방 / 메시지 페이지네이션

### 7. 경매 (추가 기능)

* 경매 게시물 관리
* 입찰 시스템 및 동시성 제어
* 낙찰 및 자동 결제 처리
